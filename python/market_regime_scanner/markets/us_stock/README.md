# US Stocks Scanner

Specialized scanner for MT5-based US equities.

## Instructions

### Scan all stocks:
```powershell
python market/us_stocks/scanner.py --all
```

### Scan single stock:
```powershell
python market/us_stocks/scanner.py --symbol AAPLm
```

### Debug with TPO Chart:
```powershell
python market/us_stocks/viz_tpo_chart.py --symbol AAPLm
```
