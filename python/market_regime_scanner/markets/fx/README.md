# Forex & Metals Scanner

Specialized scanner for MT5-based currency pairs and precious metals.

## Instructions

### Scan all symbols:
```powershell
python market/forex/scanner.py --all
```

### Scan single symbol:
```powershell
python market/forex/scanner.py --symbol XAUUSDm
```

### Debug with TPO Chart:
```powershell
python market/forex/viz_tpo_chart.py --symbol XAUUSDm
```
