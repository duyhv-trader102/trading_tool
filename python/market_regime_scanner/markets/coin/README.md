# Coin Market Scanner

Specialized scanner for MT5-based cryptocurrency pairs.

## Instructions

### Scan all coins:
```powershell
python market/coin/scanner.py --all
```

### Scan single coin:
```powershell
python market/coin/scanner.py --symbol BTCUSDm
```

### Debug with TPO Chart:
```powershell
python market/coin/viz_tpo_chart.py --symbol BTCUSDm
```
