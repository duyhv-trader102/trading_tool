# VN Stocks Scanner

Specialized scanner for VNStock-based market data.

## Instructions

### Scan VN30:
```powershell
python market/vnstock/scanner.py --vn30
```

### Scan VN100:
```powershell
python market/vnstock/scanner.py --vn100
```

### Scan single stock:
```powershell
python market/vnstock/scanner.py --symbol VNM
```

### Debug with TPO Chart:
```powershell
python market/vnstock/viz_tpo_chart.py --symbol VNM
```
